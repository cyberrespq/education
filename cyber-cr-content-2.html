เทคโนโลยีที่สนับสนุน Cyber Crime

   Computer Crime หรือ อาชญากรรมทางคอมพิวเตอร์ เป็นภัยคุกคามที่เพิ่มขึ้นในยุคดิจิทัล โดยมีเทคโนโลยีหลายประเภทที่ช่วยเพิ่มโอกาสให้เกิดการกระทำผิดเหล่านี้มากขึ้น เทคโนโลยีดังกล่าวมีทั้งข้อดีในด้านการพัฒนาเศรษฐกิจและสังคม แต่ในทางกลับกันก็ถูกใช้เพื่อการกระทำผิดกฎหมายในลักษณะต่างๆ โดยมาในรูปแบบได้ต่อไปนี้

   1. Social Media
   เป็นเครื่องมือที่ทรงพลังสำหรับการสื่อสารและการเชื่อมต่อผู้คนทั่วโลก แต่ก็เป็นช่องทางสำคัญที่สนับสนุนการก่ออาชญากรรมไซเบอร์ในรูปแบบต่างๆ เช่น
    • การปลอมตัว (Impersonation) อาชญากรสามารถสร้างโปรไฟล์ปลอมเพื่อหลอกลวงข้อมูลส่วนบุคคล เช่น ชื่อบัญชีธนาคารหรือรหัสผ่าน
    • Phishing และ Scamming มีการส่งลิงก์หลอกลวงหรือข้อความที่กระตุ้นให้เป้าหมายเปิดเผยข้อมูลสำคัญ
    • Cyberbullying และการล่อลวง (Grooming) การใช้แพลตฟอร์มเพื่อคุกคามหรือสร้างความสัมพันธ์เพื่อหาประโยชน์

   Social Media ยังเป็นช่องทางที่ช่วยให้อาชญากรแพร่กระจายข้อมูลปลอม (Fake News) เพื่อหวังผลทางการเมืองหรือการเงิน

• • •

   2. Cryptocurrency
   หรือสกุลเงินดิจิทัล เช่น Bitcoin และ Ethereum ถูกนำมาใช้ในอาชญากรรมไซเบอร์ในหลายรูปแบบ จากสาเหตุดังต่อไปนี้
    • การซื้อขายผิดกฎหมาย Dark Web เป็นตลาดสำหรับการซื้อขายสินค้าผิดกฎหมาย เช่น ยาเสพติด อาวุธ และข้อมูลบัตรเครดิตที่ถูกขโมย
    • บริการทางอาชญากรรม มีการว่าจ้างแฮกเกอร์หรือจ้างบุคคลให้ทำกิจกรรมผิดกฎหมาย เช่น การโจมตีเว็บไซต์ (DDoS)
    • การสื่อสารที่ปลอดภัย อาชญากรมักใช้ Dark Web ในการวางแผนและสื่อสารโดยหลีกเลี่ยงการตรวจจับจากเจ้าหน้าที่

   แม้ว่าจะมีการนำ Dark Web ไปใช้ในทางที่ถูกกฎหมาย เช่น การรักษาความลับของนักข่าว แต่ในหลายกรณีก็ถูกใช้อย่างแพร่หลายในอาชญากรรมไซเบอร์

• • •

   3. Dark Web
    เป็นส่วนหนึ่งของอินเทอร์เน็ตที่ไม่สามารถเข้าถึงได้ผ่าน Search Engine ทั่วไป และต้องใช้ซอฟต์แวร์พิเศษ เช่น Tor Browser
    • Ransomware เป็นช่องทางที่อาจทำให้เกิดการติด Ransomware เข้าสู่ระบบคอมพิวเตอร์โดยไม่ได้รับอนุญาต โดยส่วนใหญ่จะเกิดจากการเข้าเว็บไซต์ที่ไม่ปลอดภัย การคลิกลิงค์โฆษณาที่เป็นอันตราย หรือการดาวน์โหลดไฟล์ driver จากอินเทอร์เน็ต
    • การทำงานแบบ Hybrid เป็นการทำงานที่ไม่ได้ล็อคหน้าจอก่อนจะลุกออกจากโต๊ะทำงาน และใช้งาน WiFi ที่ไม่รู้จัก ซึ่งมีความเสี่ยงที่จะถูกแฮกข้อมูล การนำคอมพิวเตอร์และอุปกรณ์ส่วนตัวที่มีความเสี่ยงมาใช้ในการทำงาน (ไม่ติดตั้ง Antivirus / ไม่ได้อัปเดต Patch / เข้าเว็บไซต์เสียง)
    • ภัยคุกคามทางสังคม เป็นการหลอกลวงโดยอาศัยหลักจิตวิทยาให้เหยื่อหลงเชื่อ เพื่อให้ได้ผลประโยชน์ตามที่แฮกเกอร์ต้องการ ภัยคุกคามทางสังคมใช้เทคนิคการกระตุ้นความอยากรู้อยากเห็นของเหยื่อ เพื่อให้เหยื่อเปิดเผยข้อมูลสำคัญ

   Social Media ยังเป็นช่องทางที่ช่วยให้อาชญากรแพร่กระจายข้อมูลปลอม (Fake News) เพื่อหวังผลทางการเมืองหรือการเงิน

• • •

   4. Digital Technology
   มีเทคโนโลยีที่ถูกคิดค้นขึ้นมาและมีบทบาทสำคัญ ได้แก่ ปัญญาประดิษฐ์ (AI) และ Internet of Things (IoT) ถูกนำมาใช้ในทางที่ดี แต่ในทางกลับกัน อาชญากรก็ใช้เทคโนโลยีเหล่านี้เพื่อประโยชน์ของตนเอง เช่น
    • AI และ Machine Learning ช่วยให้การโจมตีทางไซเบอร์ เช่น Phishing และ Malware มีความแม่นยำและซับซ้อนมากขึ้น
    • IoT Devices อุปกรณ์ IoT ที่ไม่ได้รับการป้องกันอย่างเหมาะสมสามารถถูกแฮ็กเพื่อเข้าถึงเครือข่ายที่ใหญ่ขึ้น หรือใช้เป็นเครื่องมือในการโจมตีแบบ DDoS
    • Deepfake การสร้างสื่อปลอมที่เหมือนจริง เช่น วิดีโอปลอมของบุคคลสำคัญ เพื่อหลอกลวงหรือทำลายชื่อเสียง

   การใช้เทคโนโลยีเหล่านี้แสดงให้เห็นถึงศักยภาพที่อาชญากรสามารถเข้าถึงและใช้ประโยชน์ในวิธีที่หลากหลาย

• • •

   เทคโนโลยีที่กล่าวถึงทั้งหมดมีบทบาทสำคัญในการสนับสนุน Cyber Crime ทั้งในด้านการสร้างโอกาสและการเพิ่มความซับซ้อนให้กับรูปแบบอาชญากรรมไซเบอร์ แม้ว่าจะมีข้อดีในด้านการพัฒนานวัตกรรมและเศรษฐกิจโลก แต่ก็จำเป็นอย่างยิ่งที่ทุกฝ่ายจะต้องร่วมมือกันในการพัฒนามาตรการเพื่อป้องกันและจัดการกับภัยคุกคามเหล่านี้ เช่น การออกกฎหมายที่เข้มงวด การให้ความรู้แก่ประชาชน และการพัฒนาเทคโนโลยีด้านความปลอดภัยเพื่อปกป้องผู้ใช้งานในยุคดิจิทัล
